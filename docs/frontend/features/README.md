# Frontend Features Detail

Folder ini untuk detail frontend per fitur.
Hanya dibuat jika fitur memang punya relasi UI/frontend yang cukup penting.
Jangan membuat file kosong untuk setiap fitur.

Nama file disarankan: `FXX_FEATURE_NAME_UI.md`

Struktur file detail frontend per fitur:
